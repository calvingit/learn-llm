## 本章核心问题

Prompt 为什么有用？

## 生活比喻

Prompt 不是魔法咒语，更像是给大模型设置说话背景和回答方向。

你说“请用小学生能懂的话解释”，它就会倾向于用简单语言。你说“请像律师一样严谨分析”，它就会倾向于使用更谨慎的表达。

## 简单解释

大模型是在判断接下来最可能生成什么样的回答。Prompt 会改变它看到的上下文，所以也会改变它接下来的生成方向。

清楚的 Prompt 通常包含四件事：

- 背景：现在是什么场景
- 目标：你想得到什么
- 限制：不要做什么、要注意什么
- 输出形式：用列表、短文、步骤还是表格

## 真实使用场景

如果你只说“帮我写一个方案”，模型会猜很多东西。如果你补充对象、目标、预算、时间和语气，它就更容易给出可用的初稿。

## 小问题

好的 Prompt 不一定复杂。关键不是写很多花哨词，而是把问题说清楚。

## 一句话总结

Prompt 的作用是：改变模型接下来最可能生成什么样的回答。
