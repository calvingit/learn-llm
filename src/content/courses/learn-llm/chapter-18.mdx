## 核心问题

Agent 和普通聊天机器人有什么区别？

普通聊天更像一问一答。Agent 则会围绕一个目标，反复经历“思考、行动、观察、再决策”的循环。

## 先建立直觉

如果你说“帮我查询今天北京天气，并根据天气推荐穿搭”，普通模型可能凭印象回答。

更像 Agent 的系统会先判断需要查天气，然后调用工具，拿到结果，再根据结果给建议。

## Agent 的最小闭环

- **目标**：用户想完成什么。
- **判断**：下一步应该查资料、调用工具，还是直接回答。
- **行动**：系统执行工具或步骤。
- **观察**：把结果交回模型。
- **停止**：判断目标是否已经完成。

## 先学最小版本

很多文章会把 Agent 讲得很复杂：记忆、计划、反思、多智能体协作。

入门阶段先抓住最小闭环就够了：模型负责决策，工具负责执行，观察结果回到模型，直到目标完成或需要人类介入。

## 一句话总结


> Agent 的核心不是会聊天，而是能围绕目标不断推进下一步。
