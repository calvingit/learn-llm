## 核心问题

为什么 Prompt 会影响答案？

因为 Prompt 不是魔法咒语，而是模型当前能看到的上下文。你给它什么背景、目标、限制和格式，它就会在这个范围里预测更可能的回答。

## 先建立直觉

如果你只说“帮我写个总结”，模型不知道你要给谁看、重点是什么、要多长、要不要列行动项。

如果你说清楚角色、任务、边界和格式，它就更容易生成可用结果。

## Prompt 的四个基本要素

- **角色和背景**：你希望模型站在哪个视角回答。
- **任务和目标**：你到底要它完成什么。
- **限制和边界**：哪些不能猜，哪些必须标注不确定。
- **输出格式**：列表、表格、JSON、邮件、提纲还是说明文。

## 好 Prompt 不一定很长

很多糟糕 Prompt 很长，但只是堆要求。好的 Prompt 是结构清楚、信息必要、边界明确。

如果任务复杂，不要把所有要求塞进一个巨长 Prompt。更好的方式是把任务拆成几步，形成工作流。

## 一句话总结


> Prompt 的作用不是施法，而是把模型带到正确的任务现场。
